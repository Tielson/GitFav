## About

Rocketseat's challenge, to create a page that can get the GitHub Users API and save the user in a list, with information on the number of followers and projects done.



Several concepts were applied in this challenge, but for the creation of the logical part and structuring of my data were:



- Object orientation.

- Inheritance in JavaScript.

- Static methods.

- Promises.

- Application flow with try, catch, throw.
